# 🛒 E-Commerce Website (HTML, CSS, JavaScript)

A modern, responsive, and feature-rich **E-Commerce Website** built using **pure HTML, CSS, and JavaScript**.
This project simulates a real-world online shopping experience with dynamic cart functionality and an attractive UI.

---

## 🚀 Features

* 🏠 **Home Page** with hero banner
* 🛍️ **Product Listing** with grid layout
* 🔍 **Search** and category filtering
* 📄 **Product Detail Page** with image gallery
* 🛒 **Add to Cart** functionality
* ➕ **Update Quantity** / ❌ **Remove Items**
* 💰 **Dynamic Total Price Calculation**
* 💾 **Cart Data stored using LocalStorage**
* 📱 **Fully Responsive Design** (Mobile, Tablet, Desktop)
* 🌙 **Dark Mode Toggle** (optional)
* ❤️ **Wishlist Feature** (optional)
* ⭐ **Product Rating UI**

---

## 🧰 Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)

---

## 🎯 Purpose

This project is designed to:

* Practice **frontend development skills**
* Build a **real-world portfolio project**
* Understand **e-commerce functionality basics**

---

## 📌 Note

This is a **frontend-only project** (no backend or payment integration included).
